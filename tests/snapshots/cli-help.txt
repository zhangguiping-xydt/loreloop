$ loreloop --help
usage: loreloop [-h] [--agent {claude,codex,opencode,co-mind}]
                {doctor,codex,opencode,claude,comind,trust,init,demo,ingest,repo,project,web,verify,begin,complete,run,check,report,harvest,knowledge}
                ...

Reverse-engineer project knowledge, apply it to coding-agent work, and return
accepted outcomes to a tamper-evident knowledge loop.

positional arguments:
  {doctor,codex,opencode,claude,comind,trust,init,demo,ingest,repo,project,web,verify,begin,complete,run,check,report,harvest,knowledge}
    doctor              check prerequisites and writable trust state
    codex               install or inspect the native Codex integration
    opencode            install or inspect the native OpenCode integration
    claude              install or inspect the native Claude Code integration
    comind              install or inspect the native co-mind integration
    trust               inspect or recover this project's local trust
    init                set up LoreLoop in this project
    demo                run the bundled five-minute knowledge loop
    ingest              reverse-engineer knowledge from a source
    repo                manage repositories in this trust domain
    project             manage the federation project registry
    web                 govern replayable Web test scenarios
    verify              verify an expectation against a live page
    begin               prepare knowledge for the coding-agent session already
                        in use
    complete            mark a prepared current-session implementation
                        complete
    run                 delegate a task in a new coding-agent process
    check               record an acceptance check for a run
    report              render the acceptance report for a run
    harvest             flow knowledge back from an accepted run
    knowledge           inspect, curate and verify knowledge entries

options:
  -h, --help            show this help message and exit
  --agent {claude,codex,opencode,co-mind}
                        coding-agent CLI used for extraction and delegated
                        work (default: claude)
